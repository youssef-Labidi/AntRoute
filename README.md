##  Overview

**AntRoute** is a comprehensive platform for developing and simulating decentralized multi-agent delivery systems inspired by swarm intelligence. It combines real-time WebSocket communication with advanced visualizations to model autonomous robot behaviors, task auctions, and adaptive pathfinding.

---

## Why AntRoute?

This project empowers developers and researchers to build, test, and visualize complex autonomous systems. The core features include:

- ** WebSocket Coordination**: Facilitates real-time communication between robots and dashboards for seamless resource management.

- ** Multi-Agent Simulation**: Implements decentralized, bio-inspired algorithms for emergent behaviors and fault tolerance.

- ** Visual Analytics**: Provides interactive visualizations of pheromone trails, agent paths, and task auctions.

- ** Modular Architecture**: Built with React, Tailwind CSS, Vite, and PostCSS for scalable, maintainable frontend development.

- ** Adaptive Pathfinding**: Supports dynamic environment modeling and resilient routing strategies.

##  Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language**: JavaScript
- **Package Manager**: npm

### Installation

Build AntRoute from the source and install dependencies:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/youssef-Labidi/AntRoute
   ```

2. **Navigate to the project directory:**
   ```bash
   cd AntRoute
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```
